== Configurations ==

= Follow the following steps for a fully functional Power Form Builder: =
1. After installation, go to "Power Forms" menu in the admin panel, You will find the builtin template "Contact Us".
1. To display form just, add plugin dynamic shortcode in an existing page or add a new page/post and write shortcode anywhere in the page/post editor.
1. After someone submit a form from the front-end, you will find an entry in backend section Entries just under the Power Forms Menu.
1. Configure SMTP settings Power Forms > Settings > SMTP if you can not find an email on form submission.
1. You can also integrate Form CAPTCHA just add Google Recpatch Keys in Power Forms > Settings > reCAPTCHA Integration Tabs and activate the recaptch using indvidual form Settings.

== Installation ==

1. Download plugin.
1. Upload `power-forms.zip` to the `/wp-content/plugins/` directory to your web server.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add plugin dynamic shortcode in an existing page or add a new page/post and write shortcode anywhere in the page/post editor.

== Frequently Asked Questions ==

= How to create a new form? =
In your WordPress admin panel, go to "Power Forms" menu and add a new form. Click/Drag & Drop Fields for create a form and then copy and paste shortcode anywhere on page/post.

= Power Form Builder saving the submission data? =
In your WordPress admin panel, go to "Power Forms > Entries" where you can find you submited entries detail.
